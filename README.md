# [W.I.P] Axer:  Project Scaffolding CLI
Axer es una aplicación CLI escrita en Rust que permite generar proyectos a partir de plantillas predefinidas, aplicando convenciones y estructuras iniciales de proyecto.

El objetivo de Axer es acelerar el arranque de nuevos proyectos, evitando configuraciones repetitivas y proporcionando una base consistente sobre la cual trabajar.
Las plantillas pueden ser opinionadas, pero no imponen una arquitectura específica; solo establecen un punto de partida claro y mantenible.
